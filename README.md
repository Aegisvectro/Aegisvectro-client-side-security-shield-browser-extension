🛡️ AegisVectro - AI Security Agent

AegisVectro is a powerful, client-side browser security extension that uses local heuristics and optional AI integration to detect phishing, malware, and social engineering attacks in real-time.

Features

🚀 Client-Side Engine: Runs locally on the browser. No external server required.

🧠 Optional AI: Users can bring their own Google Gemini API Key for advanced threat analysis.

🔍 Deep Inspection: Analyzes HTML, DOM mutations, and hidden fields.

✅ Official Site Verification: Whitelisting for trusted domains (e.g., aegisvectro.com).

🚫 Dark Pattern Detection: Identifies fake countdowns and scarcity tactics.

Installation (Developer Mode)

Clone this repository.

Open Chrome and navigate to chrome://extensions/.

Enable Developer mode (top right).

Click Load unpacked.

Select the folder containing these files.

Privacy

This extension analyzes page content locally. If AI mode is enabled, data is sent directly from the user's browser to Google's Gemini API. No data is sent to AegisVectro servers.
